# Face Image Capture & PDF Generator (Python)

This project is a simple Python-based automation system that captures images using a webcam and converts them into a single PDF file. It is useful for attendance systems, identity verification, data collection, and basic computer vision practice.

The project consists of **two Python scripts**:

1. **Image Capture Script**
   - Captures 200 images of a person using a webcam
   - Stores images inside a folder named after the person

2. **PDF Creation Script**
   - Reads all images from the folder
   - Converts them into a single PDF file
   - Saves the PDF inside the same folder
